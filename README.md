# ApnaDarzi.pk - Online Custom Tailoring Platform

## Project Overview

ApnaDarzi.pk is a complete, functional website for an online custom tailoring service based in Pakistan. The platform allows customers to order custom-stitched clothes, including men's shalwar kameez and women's custom dresses.

## Features

✅ **Complete Website with 6 Pages:**
- Homepage with hero section and featured products
- Shop page with product catalog and category filtering
- Product detail page with customization form
- Custom order page for 100% custom designs
- About Us page
- Contact Us page

✅ **Key Functionalities:**
- Responsive design (mobile, tablet, desktop)
- Form validation and submission
- Product filtering by category
- Dynamic product loading
- Image upload support
- Success modals
- Mobile-friendly navigation

✅ **Design Features:**
- Clean, modern, minimal design
- Pakistani fashion aesthetic
- Easy navigation
- Professional color scheme
- User-friendly forms

## How to Use

1. **Open the Website:**
   - Simply open `index.html` in any modern web browser
   - All pages are linked and functional

2. **Navigate the Site:**
   - Use the top navigation menu to move between pages
   - Click on products to view details and customize
   - Use "Shop" to browse all products
   - Use "Custom Order" for completely custom designs

3. **Place an Order:**
   - Browse products on the Shop page
   - Click "Customize & Order" on any product
   - Fill out the customization form
   - Submit your order (shows success message)

4. **Custom Order:**
   - Go to "Custom Order" page
   - Describe your dream outfit
   - Upload reference images
   - Enter measurements or request assistance
   - Submit your request
